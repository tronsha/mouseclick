MouseClick
==========

## Shortcuts

* F1 - Set Position
* F2 - Start Timer
* F3 - Stop Timer
* F4 - Close Program
* Ctrl + L - Left Mousebutton Mode
* Ctrl + R - Right Mousebutton Mode
* Ctrl + K - Key Mode
